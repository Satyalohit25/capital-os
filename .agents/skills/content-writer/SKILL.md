# Content Writer Heuristics (Orwellian Rules)

Keep all user interface copy, labels, and text communication direct, clear, and simple.

## 1. Core Rules
1.  **No Dead Figures**: Never use a metaphor, simile, or other figure of speech which you are used to seeing in print.
2.  **Short Words First**: Never use a long word where a short one will do.
3.  **Cut Excess Words**: If it is possible to cut a word out, always cut it out.
4.  **Use Active Voice**: Never use the passive where you can use the active.
5.  **Avoid Jargon**: Never use a foreign phrase, a scientific word, or a jargon word if you can think of an everyday English equivalent.
6.  **Human Touch**: Break any of these rules sooner than say anything outright barbarous.

## 2. UI Writing Implementation
*   **Form Field Labels**: Use plain everyday nouns. (e.g., use "Income name" instead of "Revenue Source Identifier").
*   **Action Buttons**: Use a clear, active verb starting the label. (e.g., use "Next Step" instead of "Navigate Forward").
*   **Explanations**: Limit sub-text to a single, direct, active sentence describing what to do.
